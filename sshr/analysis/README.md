# Analysis Scripts

One-off diagnostic, tuning, and investigation scripts used during development.

These scripts are **not maintained as primary entry points** and may not be up to date with the latest API changes. They are kept for reference and reproducibility of past analyses.

## Scripts

- `approach_paper.py` — Paper approach analysis
- `diagnose_candidates.py` — Parallelotope candidate diagnosis
- `tiebreak_study.py` — Tie-breaking strategy study
- `analyze_sshr_h.py` — SSHR-H algorithm analysis

## Usage

Run directly with the project Python interpreter:
```bash
cd sshr
/opt/anaconda3/envs/mcts-qoracle/bin/python analysis/<script>.py
```
