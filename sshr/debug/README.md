# Debug Scripts (Historical)

This directory contains historical debugging and development scripts from various stages of the project. They are **not actively maintained** and may reference outdated APIs or experimental features.

## Policy

- **Read-only reference**: Treat these scripts as read-only references rather than code to refactor or extend.
- **No guarantees**: Scripts here may not run correctly with the current codebase.
- **Not for CI**: These are not part of the test suite or official reproduction pipeline.

## Notable Scripts

- `test_gurobi.py` — Gurobi solver smoke test
- `check_npn.py` / `npn_reps.py` — NPN equivalence class verification
- `debug_sshr_h*.py` — SSHR-H debugging variants
- `profile_n4.py` — n=4 profiling
