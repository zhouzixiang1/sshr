## Project Overview

This repository contains **two independent research papers** on quantum circuit synthesis:

### Paper 1 вҖ” SSHR Oracle Synthesis (`sshr/`)

Reproduction and extension of **"CNOT Oriented Synthesis for Small-Scale Boolean Functions Using Spatial Structures of Parallelotopes"** (Zheng et al., 2025). Implements quantum circuit oracle synthesis for Boolean functions (n вүӨ 8 variables) using geometric parallelotope structures, minimizing T-count and CNOT count.

### Paper 2 вҖ” Sparse Quantum State Preparation (`tex/`)

An original Chinese paper: **"й’ҲеҜ№зЁҖз–ҸйҮҸеӯҗжҖҒеҲ¶еӨҮзҡ„жңҖдјҳзәҝи·Ҝз ”з©¶"** (Optimal Circuit Research for Sparse Quantum State Preparation). Proposes a polynomial-time algorithm for sparse quantum state preparation without ancilla qubits, with circuit locality optimization for NISQ devices (Heavy-Hex, Square topologies). Status: paper written, LaTeX source in `tex/`.

The reference paper PDF for Paper 1 lives in `papers/`.

## Update Semantics (critical)

| Algorithm | Rule | Semantic |
|---|---|---|
| SSHR-H / SSHR-I | `A вҶҗ A вҠ• vertices(P)` | XOR вҖ” off-set can re-enter A |
| SSHR-MCTS / Beam | `A вҶҗ A вҲ’ vertices(P)`, `vertices(P) вҠҶ A` | Monotone shrink |

Do not mix these semantics when modifying synthesis algorithms.
